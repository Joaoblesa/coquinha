O que é um repositório no Git e qual a sua importância? 
e onde voce guarda seus projetos e seus codigos

Qual a função do Git Bash? 
é um aplicativo que permite usar o Git no Windows através de comandos em linha

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
git add e pra salvar seus arquivos e git add . e pra enviar seus arquivos para o repositorio

Por que utilizamos o ponto (.) no comando git add .?
para adicionar tudo

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
e para descrever oque ta salvando especificamente

Qual a função do comando git push?
para enviar alteracoes 

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
para deixar tudo que voce fez no projeto guardado e comprovado

Explique a função das tags HTML e como elas estruturam uma página.
define a estrutura e o significado do conteudo de uma pagina web

O que são atributos em uma tag HTML e qual a sua função?
sao informacoes adicionais que definem o comportamento de um elemento html

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
e um facilitador de escritas de codigos

O que é o box-sizing no CSS e por que usamos ele no reset inicial?
e pra definir a margin

Por que é importante utilizar tags semânticas no HTML?
tornam os sites mais acessíveis, otimizados para mecanismos de busca e mais fáceis de manter

O que são media queries no CSS e qual a sua função?
blocos de código CSS que definem estilos de acordo com características do dispositivo, como largura, altura e cor

Qual a função do GitHub Pages?
para deixar a pagina online

O que é o Flexbox no CSS e para que ele é utilizado?
é um modelo de layout do CSS que permite organizar elementos em um contêiner, distribuindo espaços entre eles

Qual a diferença entre justify-content e align-items em um container Flexbox?
justify-content controla o alinhamento dos itens ao longo do eixo principal, enquanto align-items controla o alinhamento ao longo do eixo transversa

Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
voce define o tanto de espaco o  elemento vai ocupar dentro do container

Explique a diferença entre flex-direction: column; e flex-direction: row;.
o column faz a divisao em colunas e o outros em linhas
